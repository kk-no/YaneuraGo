# YaneuraGo
Go adapter to YaneuraOu

## Command
```
$ go run cmd/yaneurago/main.go
$ isready
$ setoption name <id> [value <x>]
$ usinewgame
$ position [sfen <sfenstring> | startpos ] moves <move1> ... <movei>
$ go [option <infinite/btime wtime/depth>] [value]
$ stop
$ quit
```
